# BOOKLIB - REST API

A Book Lib REST API builded by spring-boot

## Authors

- Harisatul Aulia [@haris.aulia404](https://github.com/Harisatul/booklib)


## Deployment
To build this project run

```bash
  mvn clean package
```


To run this project run

```bash
  cd target
  java -jar booklib-0.0.1-SNAPSHOT.jar
```
